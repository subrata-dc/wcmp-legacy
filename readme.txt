A free plugin compatible with WC Marketplace 3.0 with all the removed settings.

== Description ==

Missing the old WCMp settings!! Here comes Legacy plugin. 

Extend you WC Marketplace where you can customize the all the vendor's capability releated to vendor's products and orders and much more!!!

== Installation ==
NOTE: WCMp Legacy Settings plugin is an extension of WC Marketplace 3.0, so the WC Marketplace 3.0 plugin must be installed and activated in your WordPress site for this plugin to work properly.


1. Download and install WCMp Legacy Settings plugin using the built-in WordPress plugin installer.
If you download WCMp Legacy Settings plugin manually, make sure it is uploaded to `/wp-content/plugins/` and activate the plugin through the `Plugins` menu in WordPress. Or follow the steps below:
Plugins > Add new > Upload plugin > Upload wcmp-legacy.zip > Install Now > Activate Plugin.
2. Configure as you see fit, under WCMp / Settings/ Legacy.
